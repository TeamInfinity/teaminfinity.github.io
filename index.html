About Infinity Kernel

Infinity Kernel is a custom kernel for Xiaomi Redmi 4A.
It's main goal is to maintain a good battery life without sacrificing performance.
Here's our XDA thread, by the way.

Murali? Fabian? What/Who are those?

Fabian and Murali are two different device tree maintainers.
Fabian includes /vendor support and uses different drivers than Murali's drivers.

If you are using a Treble ROM or the LineageOS 15.1 that is published on XDA, sorry. We do not have a kernel for you for now.
Any other ROM/If you aren't sure, try the Murali one first. If it doesn't boot, try the Fabian one.

Steps to flash:

1. Clear Kernel Auditor's data. (We recommend not using it at all on the stable channel.)
2. Wipe "Dalvik Cache" and "Cache".
3. Flash kernel.
4. Reboot.

Stable Channel

This channel is recommended for most users.
It's supposed to be a "Flash and Forget" version.
All our recommended settings are set default already so no need to use Kernel Auditor.
